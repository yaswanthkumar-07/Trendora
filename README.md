# Trendora — Digital Fashion Storefront

🏆 1ST PRIZE WINNING HACKATHON PROJECT

Trendora is a fully functional digital fashion storefront developed during a hackathon. The project provides a smooth e-commerce experience with dynamic cart management, automated GST calculation, and real-time order handling.

## Features

- Dynamic shopping cart system
- Automated GST calculation
- Real-time order management
- Responsive UI design
- Product showcase interface
